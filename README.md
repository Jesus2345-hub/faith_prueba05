# 🎮 Prueba Numero 05

## 📝 Descripción
Un juego hecho con SFML 3.0.2 y C++17.

## 🛠️ Requisitos
- Visual Studio 2022
- SFML 3.0.2 (incluida en `/external`)

## 🚀 Cómo compilar
1. Abrir `MiJuegoSFML3.sln`
2. Seleccionar plataforma **x64**
3. Compilar: `Ctrl + Shift + B`
4. Ejecutar: `F5`

## 📁 Estructura
- `src/` - Código fuente
- `include/` - Cabeceras
- `assets/` - Recursos
- `external/SFML-3.0.2/` - SFML
